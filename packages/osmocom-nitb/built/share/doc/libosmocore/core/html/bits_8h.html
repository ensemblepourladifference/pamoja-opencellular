Detailed Description

Osmocom bit level support code.

NOTE on the endianess of pbit_t: Bits in a pbit_t are ordered MSB first, i.e. 0x80 is the first bit. Bit i in a pbit_t array is array[i/8] & (1<<(7-i%8))
